# actions

Centralized reusable GitHub Actions workflows for WorkOS SDKs. Update a workflow once here and the change propagates to all downstream repos.

## Available workflows

### `block-generated-edits.yml`

Closes any PR that hand-edits an `oagen`-generated file outside its
`@oagen-ignore-start` / `@oagen-ignore-end` regions, and comments explaining
why. A file is treated as generated when its header carries the marker
`This file is auto-generated by oagen. Do not edit.`; the detector keys off
the marker text, so it works for every SDK language (C#, Kotlin, TypeScript,
PHP, Python, Ruby, Go, Rust, Elixir, …). PRs from the SDK Automation Bot and
PRs labeled `autogenerated` are exempt.

```yaml
name: Block Generated File Edits

on:
  pull_request_target:
    # No "reopened": a maintainer reopening to override the bot should not be
    # immediately re-closed. A follow-up push (synchronize) re-checks.
    types: [opened, synchronize]

jobs:
  block-generated-edits:
    uses: workos/actions/.github/workflows/block-generated-edits.yml@main
    secrets: inherit
```
